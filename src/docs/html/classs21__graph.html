Detailed Description

A class to represent a graph.

The graph can be loaded from a file and exported to a DOT format file. It supports different types of graphs (weighted/unweighted, directed/undirected).

void s21_graph::LoadFromFile ( std::string & filename)

Loads a graph from a file.

The file format should be: First line: number of vertices. Second line: graph type (0: unweighted undirected, 1: unweighted directed, 2: weighted undirected, 3: weighted directed). Subsequent lines: adjacency matrix.

Parameters
filenameThe path to the file containing the graph data.
Exceptions
std::runtime_errorif the file cannot be opened or if the file format is invalid.
